• #1 独坐敬亭山

    众鸟高飞尽，
    孤云独去闲。

  • #2 鸟鸣涧

    人闲桂花落，
    夜静春山空。

  • #3 西洲曲

    南风知我意，
    吹梦到西州。

  • #4 问刘十九

    晚来天欲雪，
    能饮一杯无。

  • #5 梅花

    遥知不是雪，
    为有暗香来。

  • #6 宿建德江

    移舟泊烟渚，
    日暮客愁新。

  • #7 行宫

    寥落古行宫，
    宫花寂寞红。

  • #8 蝉

    居高声自远，
    非是藉秋风。

  • #9 杂诗

    及时当勉励，
    岁月不待人。
